# Trophero Chess Club Website v2

This is the public HTML website for Trophero Chess Club.

## Repository
https://github.com/ppsaheed/tro

## Deployment
The website is hosted at tropherochess.com/public_html

## Technologies Used
- HTML
- CSS (Bootstrap, custom styles)
- JavaScript (jQuery, various plugins)
- PHP (for contact form)

## File Structure
- index.html - Home page
- about-trophero-chess-club.html - About page
- our-courses.html - Courses page
- gallery.html - Gallery page
- contact-us.html - Contact page
- faq.html - FAQ page
- assets/ - CSS, JS, images, fonts
- Various policy pages (privacy, terms, refund)

## Setup
1. Clone the repository
2. Open index.html in a browser or deploy to web server

## Contact
For inquiries, visit the contact page or email through the website.
